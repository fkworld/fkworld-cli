# fkworld-cli

脚手架。

## 使用方式

```bash
# 全局安装后使用
npm i -g fkworld-cli
fkworld-cli create

# npx 直接使用
npx fkworld-cli create
```

## 特点

1. 提供完整的工程化方案，包括：husky,commitlint,lint-staged,eslint,prettier,ls-lint

## 命令

### `fkworld-cli create`

通过以下模版创建：

1. pnpm + react
2. pnpm + react + monorepo

### `fkworld-cli add`

增加工程化模块。还在开发中...
